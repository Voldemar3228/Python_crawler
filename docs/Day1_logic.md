##  Логика Day1
### Основные изменения
В рамках работы первого дня были сделаны:
1. Базовый класс **AsyncCrawler** с использованием **ClientSession**.
2. Добавлена обработка ошибок.
3. Сделано логирование.
4. Тестирование проводилось с помощью **pytest**:
   - созданы фикстуры для тестов (тестовый сервер и переопределение логгера).
   - написаны несколько асинхронных тестов **@pytest.mark.asyncio**.
 
### Дополнительные изменения
1. Использовал семофор для ограничения кол-ва потоков.
2. В класс **AsyncCrawler** сделал доступным передачу параметра таймаута, чтобы сделать тестирование более гибким.
3. В асинхронную функцию **fetch_urls** передал список кортежей, а не словарь, чтобы при обращении к одному и тому же сайту значения не перезаписывались и их общее количество не терялось.
